# Plugin Development Guide

This guide will help you get started with developing plugins for Baize.

## Getting Started

1. Install the plugin SDK
2. Create a new plugin using the CLI tool
3. Implement your plugin logic
4. Test and debug your plugin

## Plugin Structure

A Baize plugin consists of:
- Plugin manifest (`plugin.json`)
- Main plugin class
- Optional UI components
- Configuration schema
