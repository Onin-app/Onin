## 简介

Baize是一个基于Tauri框架开发的应用程序，结合了Rust后端和Svelte前端技术栈。本指南将详细介绍如何有效地调试和记录应用程序的日志，帮助开发者快速定位和解决问题。

## Tauri日志系统配置

### 基础日志配置

Baize应用已经集成了`tracing`库作为主要的日志记录系统。在`src-tauri/src/lib.rs`中可以看到以下配置：

```rust
#[cfg_attr(mobile, tauri::mobile_entry_point)]
pub fn run() {
    tracing_subscriber::fmt()
        .with_span_events(FmtSpan::FULL)
        .try_init()
        .ok();
    // ...
}
```

这个配置启用了完整的span事件跟踪，提供了详细的上下文信息。

### Cargo.toml中的日志依赖

在`src-tauri/Cargo.toml`中可以看到相关的依赖配置：

```toml
[dependencies]
tracing = "0.1"
tracing-subscriber = { version = "0.3", features = ["env-filter"] }
```

### 日志级别配置

可以通过环境变量来控制日志级别：

```bash
# 开发模式 - 显示所有日志
RUST_LOG=debug cargo tauri dev

# 生产模式 - 只显示错误和警告
RUST_LOG=error cargo tauri build

# 特定模块的日志
RUST_LOG=baize_lib=debug,cargo_tauri=info cargo tauri dev
```

### 自定义日志格式

可以在`lib.rs`中添加自定义格式化选项：

```rust
tracing_subscriber::fmt()
    .with_max_level(tracing::Level::DEBUG)
    .with_target(false)
    .with_timer(tracing_subscriber::fmt::time::ChronoLocal::rfc3339())
    .compact()
    .try_init()?;
```

## 常见问题排查

### 窗口无法显示

**症状**：应用程序启动但主窗口不显示

**排查步骤**：

1. **检查窗口配置**：
```rust
// 在tauri.conf.json中检查窗口设置
{
  "windows": [
    {
      "width": 800,
      "height": 400,
      "alwaysOnTop": true,
      "center": true,
      "transparent": true
    }
  ]
}
```

2. **验证窗口事件监听**：
```rust
// 检查window_manager.rs中的setup_window_events函数
window.on_window_event(move |event| match event {
    tauri::WindowEvent::Focused(true) => {
        println!("Window focused event received");
        // 注册快捷键
    }
    tauri::WindowEvent::Focused(false) => {
        println!("Window lost focus event received");
        // 处理隐藏逻辑
    }
    _ => {}
});
```

3. **检查CSS样式**：
```css
/* 确保窗口元素有正确的样式 */
.window-container {
    width: 100%;
    height: 100%;
    background-color: transparent;
    position: relative;
}
```

### 快捷键不工作

**症状**：注册的快捷键无法触发

**排查步骤**：

1. **检查权限**：
```rust
// macOS权限检查
#[cfg(target_os = "macos")]
{
    if !check_accessibility_permissions() {
        eprintln!("Warning: Accessibility permissions not granted");
        eprintln!("Please grant accessibility permissions in System Preferences");
    }
}
```

2. **验证快捷键格式**：
```rust
// 正确的快捷键格式示例
const CLOSE_WINDOW_SHORTCUT_STR: &str = "escape";
// 或者组合键
const CUSTOM_SHORTCUT_STR: &str = "ctrl+shift+p";
```

3. **调试快捷键处理**：
```rust
// 在handle_global_shortcut中添加详细日志
pub fn handle_global_shortcut(
    app: &AppHandle,
    shortcut: &Shortcut,
    event: GlobalShortcutPluginState,
) {
    println!("Shortcut event received: {:?}, state: {:?}", shortcut, event);
    
    let shortcut_str = shortcut.to_string();
    let normalized = normalize_shortcut_string(&shortcut_str);
    println!("Normalized shortcut: {}", normalized);
    
    // 查找匹配的快捷键
    let matching_shortcut = shortcuts.iter().find(|s| {
        let stored = normalize_shortcut_string(&s.shortcut);
        println!("Comparing with stored: {} -> {}", s.shortcut, stored);
        stored == normalized
    });
}
```

### 插件加载失败

**症状**：插件无法正常加载或执行

**排查步骤**：

1. **检查插件目录结构**：
```
plugins/
├── my-plugin/
│   ├── manifest.json
│   ├── index.html
│   └── assets/
```

2. **验证manifest.json格式**：
```json
{
  "id": "my-plugin",
  "name": "My Plugin",
  "version": "1.0.0",
  "description": "Plugin description",
  "entry": "index.html",
  "commands": [],
  "permissions": []
}
```

## 日志分析指南

### 结构化日志格式

Baize应用使用结构化的日志格式，便于分析和过滤：

```
[时间戳] [级别] [模块] [消息]
```

示例：
```
[2024-01-15T10:30:45Z INFO baize_lib] Window focused event received
[2024-01-15T10:30:45Z DEBUG baize_lib::window_manager] Acquiring window close lock
[2024-01-15T10:30:45Z ERROR baize_lib::shortcut_manager] Failed to register shortcut: Permission denied
```

### 关键日志类别

1. **窗口管理**：
   - `WindowEvent::Focused`
   - `window.show()` / `window.hide()`
   - 窗口状态变化

2. **快捷键管理**：
   - `Shortcut registration/unregistration`
   - `handle_global_shortcut`
   - 权限检查

3. **插件系统**：
   - `Plugin loading`
   - `Plugin execution`
   - `Plugin protocol handling`

4. **错误处理**：
   - `eprintln!` 输出的错误信息
   - `Result` 类型的错误传播

### 日志过滤技巧

```bash
# 只显示错误和警告
RUST_LOG=error,warning cargo tauri dev

# 只显示特定模块的日志
RUST_LOG=baize_lib::window_manager cargo tauri dev

# 显示所有调试信息
RUST_LOG=debug cargo tauri dev

# 结合grep过滤特定内容
RUST_LOG=debug cargo tauri dev | grep "shortcut\|window\|plugin"
```
