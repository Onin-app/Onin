# 基础用法

## 唤醒 Onin

默认快捷键为 `Alt+Space`（可在设置中自定义）。按下快捷键后，搜索框出现在屏幕中央，再次按下或按 `Esc` 可隐藏。

## 搜索指令

在搜索框中输入关键词，Onin 会自动匹配：

- **应用程序**：输入应用名称，如 `code`、`chrome`
- **文件搜索**：输入 `file`、`files`、`find`、`search` 或 `文件搜索`，再搜索本机文件和文件夹
- **内置扩展**：如 `calc`（计算器）、`clip`（剪贴板）、`emoji`（Emoji 搜索）
- **插件指令**：已安装插件注册的指令

## 键盘操作

| 按键            | 功能                         |
| --------------- | ---------------------------- |
| `↑` / `↓`       | 上下选择指令                 |
| `Enter`         | 执行选中指令                 |
| `Esc`           | 关闭窗口                     |
| `Ctrl+,` / `⌘,` | 打开设置                     |
| `Tab`           | 进入指令详情（部分指令支持） |

## 智能排序

Onin 会根据你的使用习惯自动调整指令顺序。使用越频繁的指令，排序越靠前。

可在设置 → 指令排序中调整排序模式：

- **智能排序**：综合使用频率和最近使用时间
- **频率优先**：按使用次数排序
- **最近使用**：按最后使用时间排序
- **默认排序**：不使用频率数据

## 带参数执行

部分指令支持带参数执行。当选中一个指令后，会出现输入框供你填写参数，然后按 `Enter` 执行。

## 文件与文本匹配

某些插件指令支持直接对选中内容（文件、图片、文本）触发。当你在系统中选中文件或文字后，打开 Onin，相关插件指令会自动出现在列表顶部。

文件搜索也支持按文件名、文件夹名、路径片段、扩展名和类型过滤查找结果。完整语法见 [内置扩展 - 文件搜索](./built-in#文件搜索)。
