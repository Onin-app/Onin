# 什么是 Onin？

**Onin** 是一款专为提升效率而生的现代化键盘启动器。深受 Raycast 和 uTools 的启发，基于 **Tauri + SvelteKit** 构建，融合了 Rust 的极致性能与现代 Web 技术的灵活性。

## 核心理念

- **键盘优先** — 一切操作皆可通过快捷键完成，不再依赖鼠标
- **极速响应** — Rust 驱动，毫秒级指令响应
- **无限扩展** — 通过插件 SDK 扩展任意能力

## 主要功能

| 功能       | 说明                           |
| ---------- | ------------------------------ |
| 应用启动   | 快速搜索并启动已安装的应用程序 |
| 文件搜索   | 全局文件名搜索，快速直达       |
| 计算器     | 输入表达式直接计算结果         |
| 剪贴板管理 | 查看和搜索剪贴板历史记录       |
| 翻译       | 选中文本一键翻译               |
| Emoji 搜索 | 搜索并复制 Emoji 到剪贴板      |
| 插件扩展   | 安装社区插件，扩展任意功能     |
