# Onin 数据备份与多端云同步指南

Onin 提供了安全、私密、完全由您自己掌控的 **WebDAV 数据备份与多端同步** 功能。通过将数据同步至您个人的云网盘（如 **坚果云**、**Nextcloud** 等），您可以在多台电脑上无缝共享快捷键、偏好设置和插件配置。

---

## 🌟 核心特色

- **完全私密与安全**：数据直接传输至您的个人网盘，不经过任何第三方服务器，彻底杜绝隐私泄露。
- **智能自启/退出同步**：
  - **开机自动拉取**：每次打开 Onin 时，会自动从云端检查并获取最新备份。
  - **退出自动保存**：每次关闭 Onin 或电脑关机时，会自动将您本地最新的修改保存至云端。
- **智能差异排除**：Onin 在备份时会非常聪明地排除您本地的专属配置（如窗口位置、剪贴板隐私和日志）。因此在不同分辨率、不同屏幕的电脑间同步时，绝不会发生窗口偏斜或错位。

## 🧭 快速开始：三步开启云同步

下面以国内最常用的 **坚果云** 为例，教您如何快速配置：

### 第一步：获取您的网盘 WebDAV 密钥

1. 电脑浏览器登录坚果云官网，进入 **“账户信息” -> “安全选项” -> “第三方应用管理”**。
2. 点击 **“添加应用”**，输入应用名称（例如 `Onin`），然后点击 **“生成密码”**。
3. 记下坚果云为您生成的那一串专属的 **“应用授权密码”**。

> [!WARNING]
> 出于安全考虑，请 **千万不要** 直接填写您的网盘主密码。请始终使用上述步骤生成的 **应用独立授权密码**。

### 第二步：在 Onin 设置中填写配置

1. 启动 Onin，进入 **“设置 (Settings)”** 界面。
2. 在侧边栏导航点击进入 **“数据同步”** (Sync Settings) 选项卡。
3. 对应填入您的配置信息：
   - **服务器地址**：坚果云地址为 `https://dav.jianguoyun.com/dav/`
   - **用户名**：您的坚果云注册邮箱
   - **密码/应用密钥**：第一步中生成的 **应用授权密码**
4. 点击 **“测试连接”**。如果提示连接成功，说明您的 Onin 已与云网盘顺利互通！

### 第三步：首次备份与恢复

- **在一号设备上**：点击 **“立即备份”**，将本地配置首次上传至云端（此时您的网盘中会多出一个 `onin/onin_backup.zip` 文件）。
- **在二号设备上**：配置好相同的 WebDAV 信息后，点击 **“立即恢复”**，即可一键把配置拉取到本地，应用会自动重载生效，无需手动重启。

## ❓ 常见问题与解答 (FAQ)

### Q1：点击“测试连接”时提示 401 错误（未授权）？

- **解答**：这是最常见的问题。请确保您在“密码/应用密钥”输入框里填入的是网盘网页端专门生成的 **“第三方应用授权密码”**，而不是您登录网盘的主密码。

### Q2：我不同电脑的屏幕分辨率不一样，同步后窗口位置会错乱吗？

- **解答**：**完全不会**。Onin 具备本地差异排除机制，在备份和还原时，会自动排除诸如“窗口在屏幕中的宽高、相对坐标位置”、“本地剪贴板历史”等设备专属数据。这保证了不同电脑的独立性和最佳显示效果。

### Q3：同步是自动的吗？我需要经常手动备份或恢复吗？

- **解答**：不需要。只要您在设置页面中勾选了 **“开机自动拉取”** 和 **“退出自动备份”**，Onin 就会在后台全自动帮您完成上传和还原，您几乎感知不到它的存在。

### Q4：网络不通或网盘服务器临时宕机时，应用会卡死吗？

- **解答**：**完全不会**。Onin 的同步模块配置了严格的 15 秒网络超时和完善的防崩溃保护。如果网络故障，它只会显示连接失败，绝不会影响您日常启动和操作 Onin。
